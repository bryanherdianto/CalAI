# CalAI

An AI-Powered Calendar Scheduler.

To reproduce the project, follow these steps:

```ps
pip install poetry
poetry install
Invoke-Expression (poetry env activate)
```

Run the backend server:

```ps
cd backend
uvicorn main:app --reload
```

Frontend:
```ps
npm run dev
```

Tools:
- FullCalendar
- React
